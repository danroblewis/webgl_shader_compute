Layer 2: Direct Buffer Access PERFORMANCE

This example demonstrates Layer 2 API: direct buffer access for performance-critical operations. Instead of downloading GPU data on every cell query, we get a reference to the entire buffer once.

Layer 2 API Demo

// Get buffer reference once (minimal GPU downloads)
const buffer = sim.getBuffer();

// Direct memory access - no GPU overhead!
const topLeft = buffer[0];
const topRight = buffer[sim.width - 1];
const center = buffer[sim.width * sim.height / 2];

// Modify and sync back
buffer[y * sim.width + x] = 1.0;
