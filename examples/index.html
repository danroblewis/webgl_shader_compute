About These Examples

Each example demonstrates different capabilities of the WebGL Compute Library:

  • Simple Operations: Examples 1-2 show basic element-wise operations
  • Complex Math: Example 3 combines multiple arithmetic operations
  • Stateful Computation: Example 4 shows how cells can access neighboring values

All computations run in parallel on your GPU, making them much faster than CPU equivalents for large datasets.
