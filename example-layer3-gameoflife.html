Conway's Game of Life LAYER 3: High-Level API

This example demonstrates the high-level convenience API (Layer 3). It's simple to use but may trigger GPU downloads for individual cell queries.

Layer 3 API Used

// High-level convenience methods
sim.setCell(x, y, 1);           // Set individual cell
sim.getCellState(x, y);         // Get individual cell (may download)
sim.step();                      // Run one generation
sim.randomize(0.3);             // Random fill (30% alive)
sim.clear();                    // Clear all cells
sim.fillRect(x, y, w, h, 1);    // Fill rectangular region

// Query methods (trigger downloads)
const alive = sim.countAlive();  // Count alive cells
console.log('Generation:', sim.generation);

✅ Pros: Simple, intuitive, safe
⚠️ Cons: Individual cell queries may be slow (GPU download overhead)
